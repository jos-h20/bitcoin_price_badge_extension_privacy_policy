Privacy Policy

Bitcoin Price Badge Chrome Extension

Effective Date: December 8, 2025

1. Data Collected (Non-Personal)

The Bitcoin Price Badge Chrome Extension (the "Extension") does not collect, store, or transmit any Personally Identifiable Information (PII) about its users.

The only data handled by this Extension is:

  • Current Bitcoin Price: The BTC-USD sell price, retrieved from a public API.
  • Last Updated Timestamp: The time the price was successfully fetched.

2. Data Usage and Storage

Usage: The non-personal data (Price and Timestamp) is used solely for the functionality of the Extension: to display the latest price on the Chrome toolbar badge and in the extension popup.

Storage: This data is stored locally within your browser using chrome.storage.local. It is never sent to the developer or any third party, nor is it synchronized with other devices. Once the Extension is uninstalled, this locally stored data is permanently deleted.

3. Third-Party Services

To function, the Extension connects to a public, third-party API to retrieve real-time Bitcoin price data.

  • Service: Coinbase API (for price retrieval).

This Extension relies on the public availability of this API. We are not responsible for the privacy practices of Coinbase.

Changes: We may update this Privacy Policy from time to time. We will notify you of any changes by posting the new Privacy Policy on this page.
